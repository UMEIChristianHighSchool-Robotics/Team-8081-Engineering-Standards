# 🧠 Core Engineering Philosophy

Team 8081 follows these principles:

### 1️⃣ Strategy Drives Design  
We never design before understanding the game.

### 2️⃣ Simplicity Scales  
Simple systems are easier to build, maintain, and debug.

### 3️⃣ Documentation Is a Deliverable  
If it is not documented, it is not finished.

### 4️⃣ Students Lead  
Mentors guide. Students execute.

### 5️⃣ Stability Wins Competitions  
Reliable robots outperform complex unstable ones.
